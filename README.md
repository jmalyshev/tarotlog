# TarotLog (Expo scaffold)

This scaffold provides a starting point for the TarotLog Expo app.

Getting started (Windows PowerShell):

1. Install prerequisites (if you don't have them):
```powershell
npm install --global expo-cli
# or use npx for local expo CLI
```

2. Install dependencies:
```powershell
cd C:\Users\julie\Documents\tarotlog
npm install
```

3. Start the dev server:
```powershell
npm run start
```

Notes
- This is a minimal scaffold. Replace placeholder components with full implementations (Reanimated-based gestures, Supabase wiring, and local DB sync).
- To build native binaries, use EAS Build (requires additional setup).
